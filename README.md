# InBank

A simple banking application frontend built using HTML, CSS, and JavaScript.

## Features

- User login interface
- Create account page
- Dashboard view
- Account balance display
- Money transfer simulation
- Transaction history
- View Java source code inside the app

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure
InBank/
│
├── inbank_frontend.html
└── README.md


## How to Run

1. Download or clone this repository.
2. Open `inbank_frontend.html` in any browser.
3. Use the login/create account features.
